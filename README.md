📊 Finance Dashboard

A modern, responsive personal finance management dashboard built with React.
Track your income, expenses, balance trends, and spending patterns with beautiful visualizations and insightful analytics.


✨ Features
📈 Dashboard
Summary cards for total income, expenses, and balance
Balance trend chart (last 6 months)
Spending breakdown (pie chart)
Color-coded categories
💳 Transactions
Complete transaction history
Advanced filters (category, type, amount)
Transaction detail modal
Fully responsive table
🎯 Insights
Financial analytics and statistics
Category-based spending insights
Trend analysis over time
🎨 User Experience
Modern UI using Tailwind CSS
Fully responsive design
Sidebar navigation
Clean and intuitive layout
🚀 Quick Start
Prerequisites
Node.js (v14+)
